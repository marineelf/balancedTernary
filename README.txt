BTCalculator v.1.0
Developed by TJ Du
11/18/2010

The 4 files included in this directory (with the exception of this readme) are the following:

1) BTCalculator.jar       -executable version of the calculator.
2) BTOperations.java      -code for the operations the calculator conducts.
3) BTOperationsTest.java  -unit testing of the calculator's operations.
4) CalcGUI.java           -code for the graphical user interface of the calculator.
